# ADRF User Guide

Welcome to the Administrative Data Research Facility (ADRF) User Guide. This comprehensive guide will help you navigate the research environment, manage your workspace, and conduct your research effectively.

## Quick Start

The ADRF provides a secure environment for working with administrative data. This documentation covers everything from basic navigation to advanced workflows.
