# Getting Started with ADRF

This guide will help you get started with the Administrative Data Research Facility (ADRF).

> 

## Prerequisites

Before you begin, ensure you have:

- A valid institutional email address
- Approval from your institution's IRB (if required)
- Completed required training modules
- Understanding of data confidentiality requirements

## Quick Start Steps

### 1. Account Creation

Create your ADRF account using your institutional credentials.

```bash
# No command needed - use web interface
# Visit: https://adrf.coleridgeinitiative.org/signup
```

### 2. Training Completion

Complete the mandatory security and confidentiality training modules:

- Data Security Fundamentals
- Disclosure Review Process
- ADRF Platform Overview

### 3. Project Setup

Once approved, set up your first project workspace.
